## Fitxers convertits a Markdown

Els continguts també s’han convertit a `content/*.md` perquè Codex els pugui llegir fàcilment.

## Regla important

Els fitxers `content/*.md` són contingut base. No s’han de reescriure de manera creativa sense demanar-ho. Les millores han de ser de claredat, estructura, SEO i adaptació web, mantenint el sentit original.
