# SEO i accessibilitat — Yogaia Nova Web 2026

## Principis SEO

Cada pàgina ha de tenir:

- un sol H1;
- H2 clars per seccions;
- meta title;
- meta description;
- slug net;
- text alternatiu a les imatges;
- CTAs amb text descriptiu;
- enllaçat intern.

## Accessibilitat

Aplicar:

- HTML semàntic (`header`, `nav`, `main`, `section`, `footer`);
- botons i enllaços diferenciats;
- `aria-label` quan el text visible no sigui suficient;
- focus visible;
- contrast correcte;
- formularis amb `label` associat;
- missatges d’error comprensibles;
- alt text descriptiu i no decoratiu.

## Imatges

Evitar alt texts genèrics com:

- “foto yoga”
- “imagen clase”
- “sala”

Preferir:

- “Grup practicant ioga a la sala principal de Yogaia, amb el jardí interior al fons.”
- “Professor ajustant una postura durant una classe de Hatha Raja Ioga.”
- “Sala principal de Yogaia amb llum natural i vistes al jardí interior.”
