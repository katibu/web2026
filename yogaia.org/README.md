# Yogaia Codex Context Pack

Paquet de context per importar al repositori local de la nova web de Yogaia i treballar amb Codex.

## Com utilitzar-lo

1. Descomprimeix aquest paquet.
2. Copia els fitxers i carpetes a l’arrel del teu projecte local.
3. Obre el projecte amb VSCodium.
4. Executa Codex des de l’arrel del projecte.
5. Demana-li que llegeixi `AGENTS.md` i `PLAN.md` abans de fer canvis.

## Fitxers principals

- `AGENTS.md` — instruccions que Codex llegirà com a context de projecte.
- `PLAN.md` — pla de desenvolupament.
- `docs/` — guies de context, arquitectura, estil, SEO i components.
- `content/` — continguts de pàgina convertits des dels ODT.
- `prompts/` — prompts reutilitzables per treballar amb Codex.

## Recomanació

Fes un commit abans d’enganxar aquest paquet al projecte, i un altre commit després. Així podràs revisar exactament què s’ha afegit.
